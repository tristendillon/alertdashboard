# ✅ To-Do List: Dashboard Components (Building Blocks)

## Basic Components
- [ ] Text Block – For descriptions, headings, notes
- [ ] Alert Tile – Displays real-time alert (e.g. red/yellow/green states)
- [ ] Status Indicator – Boolean or on/off state (e.g., system up/down)
- [ ] Metric Display – A single number or value (e.g., CPU %, response time)
- [ ] Chart Component
  - [ ] Line Chart
  - [ ] Bar Chart
  - [ ] Pie Chart
  - [ ] Area Chart
- [ ] Table/Log Viewer – Tabular data display, like recent errors or events
- [ ] Image/Logo Embed – Static images or icons (branding/custom UX)

## Advanced Components
- [ ] Graph Panel with Thresholds – Alert when crossing limits
- [ ] Interactive Filters – Date range pickers, dropdowns for filtering dashboard data
- [ ] Iframe Embed – For embedding external tools or dashboards
- [ ] Map Component – Geolocation-based alerting or metrics
- [ ] Custom Code Block – For power users to inject JS or query snippets
